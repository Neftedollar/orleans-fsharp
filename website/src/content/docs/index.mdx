## Features

<CardGrid>
  <Card title="grain { } CE" icon="pencil">
    Define grains declaratively with 31 computation expression keywords. State machines, handlers, lifecycle — all in idiomatic F#.
  </Card>
  <Card title="siloConfig { } CE" icon="setting">
    Configure clustering, storage, streaming, TLS, dashboard, health checks — 39 keywords covering every Orleans provider.
  </Card>
  <Card title="eventSourcedGrain { } CE" icon="document">
    Event sourcing with pure apply/handle functions. JournaledGrain bridge handles persistence.
  </Card>
  <Card title="800+ Tests" icon="approve-check">
    718 unit + 86 integration tests. Property-based testing with FsCheck. GrainArbitrary with TypeShape.
  </Card>
</CardGrid>

## Quick Start

```bash
dotnet new install Orleans.FSharp.Templates
dotnet new orleans-fsharp -n MyApp
cd MyApp
dotnet build && dotnet test
```
